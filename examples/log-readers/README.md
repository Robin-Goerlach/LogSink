# Log Reader Examples

Dieser Ordner enthält kleine Beispiele, die Logmeldungen aus LogSink lesen.

## Warum Reader, obwohl es schon den Java-Viewer gibt?

Der Java-Viewer ist die komfortable grafische Anwendung.

Die Reader-Beispiele sind kleine technische Prüfwerkzeuge. Sie helfen, schnell zu testen:

```text
Service erreichbar?
JSON kommt zurück?
Neue Meldung nach POST sichtbar?
```

## Aktuelle Beispiele

```text
curl/
```

Später können weitere Reader folgen, falls sie für Tests oder Dokumentation nützlich sind.
